# 解决 MongoDB 无权限执行命令

## 1 MongoDB not authorized on admin to execute command { replSetGetStatus: 1.0 }

```shell-script
$ rs.status()
{
    "ok" : 0,
    "errmsg" : "not authorized on admin to execute command { replSetGetStatus: 1.0 }",
    "code" : 13
}
```

给 `admin` 用户赋予集群管理的权限：

```shell-script
$ db.grantRolesToUser("admin", ["clusterAdmin"])
```
