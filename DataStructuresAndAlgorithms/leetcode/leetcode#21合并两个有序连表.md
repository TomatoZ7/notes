# leetcode 第二十一题：合并两个有序的链表

将两个升序链表合并为一个新的 升序 链表并返回。新链表是通过拼接给定的两个链表的所有节点组成的。 

### 示例1

![image](https://github.com/TomatoZ7/notes-of-tz/blob/master/images/leetcode%2320.gif)

```
输入：l1 = [1,2,4], l2 = [1,3,4]
输出：[1,1,2,3,4,4]
```

### 示例2
```
输入：s = "(]"
输出：false
```

### 示例3
```
输入：s = "([)]"
输出：false
```
