## 2.冒泡排序

### 2.1 算法实现

#### 2.1.1 Golang

```go
func BubbleSort(arr []int) {
	length := len(arr)

	for i := 0; i < length; i++ {
		// Bubble Flag
		hasBubble := false
		for j := 0; j < length - 1 - i; j++ {
			if arr[j+1] < arr[j] {
				// Bubble
				temp := arr[j+1]
				arr[j+1] = arr[j]
				arr[j] = temp
				// Flag
				hasBubble = true
			}
		}

		if !hasBubble {
			break
		}
	}
}
```
