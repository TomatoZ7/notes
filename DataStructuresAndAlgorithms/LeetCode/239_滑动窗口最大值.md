# LeetCode 第 239 题：滑动窗口最大值

给定一个数组 `nums` 和滑动窗口的大小 `k`，请找出所有滑动窗口里的最大值。

**示例 1**

```
输入: nums = [1,3,-1,-3,5,3,6,7], 和 k = 3
输出: [3,3,5,5,6,7] 
解释: 

   滑动窗口的位置        最大值
-------------------    -----
[1 3 -1] -3 5 3 6 7    3
1 [3 -1 -3] 5 3 6 7    3
1 3 [-1 -3 5] 3 6 7    5
1 3 -1 [-3 5 3] 6 7    5
1 3 -1 -3 [5 3 6] 7    6
1 3 -1 -3 5 [3 6 7]    7
```

**示例 2**

```
输入：nums = [1], k = 1
输出：[1]
```

**提示**

+ `1 <= nums.length <= 10^5`

+ `-10^4 <= nums[i] <= 10^6`

+ `1 <= k <= nums.length`
