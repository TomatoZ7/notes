# 标准化（Standardization / Z-score）

## 定义

把数据转换成 **均值为 0、标准差为 1** 的分布。

**标准化不是为了“看数值大小”，而是为了“比较偏离程度”。**

## 公式

$$
x' = \frac{x - \mu}{\sigma}
$$

其中：

- μ：均值（mean）
- σ：标准差（std）

## 举例说明

### 一维数据，解释计算流程

假设我们只有 1 个特征，比如「像素强度」或「温度」：

```
原始数据 x:
[10, 12, 14, 16, 18]
```

1. 计算均值 μ 

```
μ = (10 + 12 + 14 + 16 + 18) / 5 = 14
```

2. 计算标准差 σ

```
(10 - 14)² = 16
(12 - 14)² = 4
(14 - 14)² = 0
(16 - 14)² = 4
(18 - 14)² = 16

方差 = (16 + 4 + 0 + 4 + 16) / 5 = 8
σ = √8 ≈ 2.83
```

3. 每个数做标准化

| 原值 x | 计算           | 标准化后 x′   |
| ---- | ------------ | --------- |
| 10   | (10−14)/2.83 | **−1.41** |
| 12   | (12−14)/2.83 | **−0.71** |
| 14   | (14−14)/2.83 | **0.00**  |
| 16   | (16−14)/2.83 | **+0.71** |
| 18   | (18−14)/2.83 | **+1.41** |

### 二维数据

假设我们有 2 个特征：

| 样本 | 身高(cm) | 体重(kg) |
| -- | ------ | ------ |
| A  | 160    | 50     |
| B  | 170    | 60     |
| C  | 180    | 70     |

标准化后（每一列各算一套 μ 和 σ）

| 样本 | 身高′   | 体重′   |
| -- | ----- | ----- |
| A  | −1.22 | −1.22 |
| B  | 0.00  | 0.00  |
| C  | +1.22 | +1.22 |

原始数据回答的是：

- “这个人有多高？”
- “这个人体重多少公斤？”

标准化后的数据回答的是：

- “这个人在同类人群中，偏离平均水平多少？”

从这些“标准化数值”中，我们能读出：

- `0` → 正好是平均水平
- `+1.22` → 高于平均 1.22 个标准差
- `−1.22` → 低于平均 1.22 个标准差

更口语化的表达就是：

- B：典型的“平均身材”
- A：明显偏矮、偏瘦
- C：明显偏高、偏重

如果某个人标准化后是：

- 身高′ = +3.5
- 体重′ = +0.2

那么可以得出：

- 身高是 极端异常值
- 体重是正常的

## 代码实现

```python
from sklearn.preprocessing import StandardScaler

scaler = StandardScaler()

X_train_std = scaler.fit_transform(X_train)
X_test_std = scaler.transform(X_test)

print("Mean:", scaler.mean_)    # 均值
print("Std:", scaler.scale_)    # 标准差
```

# 归一化（Normalization / Min-Max Scaling）

## 定义

把不同量纲、不同取值范围的数据，按某种规则缩放到一个统一的范围内，常见是 [0, 1] 或 [-1, 1]，但不改变数据之间的相对大小关系。

归一化解决了什么问题？

防止“大数值特征支配模型”。在 KNN / KMeans / SVM / 神经网络 中的距离公式：

$$
d(x, y) = \sqrt{\sum (x_i - y_i)^2}
$$

如果身高差 = 10，体重差 = 2，那么身高几乎决定一切。

归一化后两个特征都在 0～1，影响更平衡。

## 公式

$$
x' = \frac{x - x_{\min}}{x_{\max} - x_{\min}}
$$

## 举例说明

原始数据：

| 样本 | 身高(cm) | 体重(kg) |
| -- | ------ | ------ |
| A  | 160    | 50     |
| B  | 170    | 65     |
| C  | 180    | 80     |

归一化计算：

- 身高（min = 160, max = 180）

| 身高  | 归一化                    |
| --- | ---------------------- |
| 160 | (160-160)/(20) = **0** |
| 170 | 10/20 = **0.5**        |
| 180 | 20/20 = **1**          |

- 体重（min = 50, max = 80）

| 体重 | 归一化             |
| -- | --------------- |
| 50 | **0**           |
| 65 | 15/30 = **0.5** |
| 80 | **1**           |

归一化后数据：

| 样本 | 身高  | 体重  |
| -- | --- | --- |
| A  | 0   | 0   |
| B  | 0.5 | 0.5 |
| C  | 1   | 1   |

## 代码实现

```python
import numpy as np
from sklearn.preprocessing import MinMaxScaler
from sklearn.model_selection import train_test_split

# 示例数据： [身高(cm), 收入(元)]
X = np.array([
    [170, 8000],
    [175, 12000],
    [180, 50000],
    [165, 7000],
    [178, 90000],
])

# 1. 划分训练 / 测试集（非常重要）
X_train, X_test = train_test_split(X, test_size=0.4, random_state=42)

# 2. 只在训练集上 fit
scaler = MinMaxScaler(feature_range=(0, 1))
X_train_scaled = scaler.fit_transform(X_train)

# 3. 测试集 / 线上数据只 transform
X_test_scaled = scaler.transform(X_test)

print("Train:\n", X_train_scaled)
print("Test:\n", X_test_scaled)
```

# 标准化 VS 归一化

| 维度       | 归一化       | 标准化     |
| -------- | --------- | ------- |
| 范围       | 固定区间（如 0～1）     | 均值 0，方差 1    |
| 是否受异常值影响 | 很敏感     | 相对不敏感  |
| 常用算法     | KNN、KMeans、神经网络 | 线性模型、SVM、PCA |
